# GitHub Portfolio Description

Phishing detection platform built with Python, Machine Learning, FastAPI, Streamlit, Tkinter, and Flutter.

Highlights:
- official `RandomForest` baseline with `16` engineered URL features
- email spoofing analysis with raw-header inspection
- desktop, web, API, and mobile delivery on top of the same detection engine
- ablation, benchmarking, calibration cases, and automated tests
- Android mobile client validated on a real device
