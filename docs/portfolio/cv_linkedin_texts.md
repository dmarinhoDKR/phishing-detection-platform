## Curriculum - Portuguese

**Projeto em destaque: Plataforma de Detecção de Phishing | Python, Machine Learning, FastAPI, Streamlit, Flutter**

- Desenvolvi uma plataforma de detecção de phishing com análise de URLs e emails, incluindo classificação em 3 níveis de risco e explicações dos principais sinais detectados.
- Criei e refinei um pipeline de machine learning com engenharia de features para URLs, estudos de ablação e benchmark de modelos para melhorar desempenho e interpretabilidade.
- Consolidei um baseline oficial com `RandomForest` e `16 features`, alcançando `F1 phishing 0.861`, `ROC AUC 0.9715` e `balanced accuracy 0.8981` em mais de `549 mil` amostras válidas.

- Desenvolvi múltiplas interfaces para o mesmo motor de análise: desktop com `Tkinter`, web com `Streamlit`, API com `FastAPI` e app mobile com `Flutter`, validado em dispositivo Android real.
- Adicionei logging estruturado, histórico de análises, casos de calibração e testes automatizados com `pytest` para proteger a baseline oficial do projeto.

## Curriculum - English

**Featured Project: Phishing Detection Platform | Python, Machine Learning, FastAPI, Streamlit, Flutter**

- Built a phishing detection platform for both URL and email analysis, including 3-level risk classification and explanations of the main risk signals.
- Designed and refined a machine learning pipeline with URL feature engineering, feature ablation studies, and model benchmarking to improve performance and interpretability.
- Consolidated an official baseline using `RandomForest` and `16 features`, achieving `0.861 phishing F1`, `0.9715 ROC AUC`, and `0.8981 balanced accuracy` on more than `549k` valid samples.
- Implemented contextual email analysis with spoofing checks, display-name versus sender-domain mismatch detection, subject/link analysis, and raw-header inspection for `Reply-To`, `Return-Path`, `SPF`, `DKIM`, and `DMARC`.
- Delivered multiple interfaces on top of the same detection engine: desktop app with `Tkinter`, web app with `Streamlit`, backend API with `FastAPI`, and a `Flutter` mobile app validated on a real Android device.
- Added structured logging, analysis history, calibration cases, and automated tests with `pytest` to protect the project’s official baseline.
