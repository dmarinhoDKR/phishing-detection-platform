# Flutter Mobile App

Cliente Flutter inicial para Android e iPhone consumindo a API em [`api_fastapi.py`](/C:/Users/danie/OneDrive/Área%20de%20Trabalho/projeto/phishing_ai/api_fastapi.py).

## Rodar

Na pasta `mobile_app_flutter`:

```powershell
flutter pub get
flutter run
```

## URL da API

O app permite editar a URL da API pela engrenagem no topo.

Sugestões:

- Android Emulator: `http://10.0.2.2:8000`
- iPhone Simulator: `http://127.0.0.1:8000`
- aparelho físico: `http://SEU_IP_LOCAL:8000`
